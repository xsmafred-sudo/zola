## STRIDE Threat Register

| Threat ID | Category | Component                 | Disposition | Mitigation Plan                                                                  |
| --------- | -------- | ------------------------- | ----------- | -------------------------------------------------------------------------------- |
| T-03-01   | S        | Agent command injection   | mitigate    | Validate and sanitize all agent commands; use allowlist for permitted operations |
| T-03-02   | T        | Agent response integrity  | mitigate    | Cryptographically sign agent responses; validate response signatures             |
| T-03-03   | I        | Tool invocation abuse     | mitigate    | Restrict tool access per agent type; implement tool usage quotas                 |
| T-03-04   | R        | Unauthorized agent access | mitigate    | Orchestrator validates user permissions before invoking agents                   |
| T-03-05   | D        | Agent resource exhaustion | mitigate    | Rate limit agent invocations per user; implement timeout controls                |
| T-03-06   | E        | Malicious agent extension | mitigate    | Code review for custom agents; sandbox external agent execution                  |

</threat_model>
